安装准备软件

  • Git for Window64

下载软件后，点击安装，然后一路下一步，安装完成。

怎么打开 Git ?

Win 平台操作

  1. 开始菜单 Git Bash
  2. 鼠标右键打开 Git Bash

注册 Github

访问：http://www.github.com/
注册你的 username和邮箱，邮箱非常重要。按照提示注册即可。
