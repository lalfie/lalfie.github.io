CG日常学习计划

Jul 29, 2017 | CG

训练方法

如何克服难点

通过局部训练来强化自己的弱项。 例如：如果眼睛画不好，就专攻眼睛；头发画得不好，就专攻头发... 像这样进行局部强化训练。克服自己的绘画弱点，需要这样专项性的反复练习。 不过仅能画好局部也并不等于就能画好角色。

如何成为高手 —— 画的时候，意识到整体，逐步培养抓住整体平衡的感觉...

培养在不同人像模式下，例如：全身像、胸像、半身像等情况时，将面部、身体、躯干、胳膊以及腿部等连接在一起的平衡感，而其中最基础的就是把握头身比。

关注不同的绘画风格，不同的平衡以及不同的画法

如果只专注于一个画家，学习一个人的画法，那么就只能成为那个人的模仿者而已。想要确立自己的风格，不但要多关注自己喜欢的画家的作品，还要积极地去学习与自己喜好相反的绘画风格，平时多观察是一种很好的学习方法。

  1. 日常速写。至少三天一张。尽量每天一到三张。用铅笔在纸上画造行。
  2. 分析整理收集图片。不同类型不同风格的画。
  3. 整理收集以后，要刻意学习里面的不同点。或者是你缺乏的点。
  4. 至少画的东西要和网上的差不多。
  5. 学习就是一种单纯的模仿。
  6. 不要急于求成。不要及浮躁。

行业瓶颈与原因

入口，每年入行学习人群100万，几年前就有统计说，动漫行业已是高饱和行业了。
但是别一头出口，每年就业，公司里仍是一将难求，万份简历里找不出能用的。就业2年以上的都是月入上万。
从入口到出口，那么多人涌入学习原画，却只有很少人学成就业。这中间巨大的瓶颈就是培训。
另一组数据，大学就业率1.5%，机构3%，大师班15%（这是游戏、影视、设计的就业统计）。
就是说，原画培训其实很不给力！！！
网络上说4个月100%高薪就业，其实是骗你的。！！而那些毕业作品是老师把好一点的改过的。
行业瓶颈的原因是：几个月怎么可能出高手。否则就业早就饱和了，人人月薪3000啦。

  • 真相：零基础学画画，要达到这份高薪就业水平，没4年是学不出来的。

记住这个真理，绝对无一例外。
根据统计结果：一般20岁之前与30岁之后这两种年龄段的人群更容易学好，因为他们‘不着急’。

同时！短期高强度训练没有用，准确的说，比低强度有用，但有些知识只有长时间的消化与沉淀才能转化！因为人短期的沉淀是有上限的。掌握任何一门技术的道理都一样，首先要多练多累积。其次，这个累积就是要时间去沉淀。画画也一样。1个月内你画一万张速写，你的进步会比画5千张的张，但绝对达不到大师水平。因为大师1个月也许只画一百张，但他画了10年。同时，他思考了3650天，而你只有30天。天壤之别，当然10年还有见闻阅历累积。去问身边有一技之长的人是否是这个道理。
除了更早开始，没有别的方法。
时间：4年

物资准备：一台电脑，一块数位板，能上网。天天呆在家里能吃饱饭就可以。学习费用倒真不高，学4年，也就4万足够了。
精神准备：长年坚持投入的热情与精力

2.学画画的捷径

什么是真正珠捷径

  1. 笨鸟先飞，更早开始，即是捷径。

这4年如果躲不过，那就早点开始。

很多技术型人才，明星，歌手，运动员。10岁或更早便开始练习生涯。才能在20岁就成人生赢家。
古代的艺术家，很多也是从小就进专业学校，才能在20岁就做出传世作品。
到了现在，怎么你就觉得可以年少玩游戏，大学谈恋爱，然后报个培训班，随便比划两下，就有了？

  1. 方法正确，不走弯路，即是捷径。

正确捷径：要4年能练成的事，每步都走对，别弄得20年都没练成。这已经是捷径了！
虚假捷径：不少投机商人利用你急功近利的心态，告诉你有4天能成的方法，然后，你的钱和精力都白花了。
4天你当然不信，4个月呢？是不是心动了？4年，少一天都不行！

总结公式：画好 = 1 方法正确 + 2 长期坚持

试错成本

开始做事前都应该衡量下做这事的投入与产出比。

CG电脑绘画行业属于新兴行业，手绘板绘画技术发展也不到20年。随着游戏影视的火热，游戏影视美术也是水涨船高，培训也越来越多。但仍处于混乱不规范的状态，很多人画了没两年就做老师，总结些套路。一知半解教人，这样很可怕，误人子弟。

这就是CG行业的培训现状：
一头是大量涌入的想学者，一头确实是坐拥高薪的成都者。
中间隔着迷雾，其中各种吸血鬼等着你，他们巴不得这迷雾永不散去，你怎么也看不清。他们做虚假广告大唱付费速成论，快速榨干你的经费，然后告诉你：你没天赋。

用钱就能快速换来技术？绝对不能！运气好或许能找个好老师得到正确方法。但这个好老师也还是会告诉你：你只能用大量的时间，精力，投入苦练，才能换来技术！
说白了再看公式：画好=方法正确+长期坚持
他们偷换概念，让你觉得有了1，就不用有2了。

###＃　试错成本最小化
毕竟行业在发展，成熟的机构也会越来越多，首先标志性的就是透明度。
其次，巨大迷雾和 主试错成本的解决方法：有！很简单，２步。

  1. 跟随正确引导分阶段学习
  2. 在1的基础上，某阶段的定点重复学习！

因为专练一点的成果反馈最即时。

举个例子：我有《九阴真经》，一天一个招式，一个月变成武林高手？那是小说的戏剧夸张手法。
阶段学习的方式是：来，先挑水，只挑手。练一个月后，如果发现确实有进步，身体变强壮了，重点是此时你也还 保持热情 ，那么下个月再练个别的（只是比作基本功，）如果一个月下来腰酸背痛，那么可能体质太差，需要调整方法，可能需要再坚持，这时候判断下一个月的方向，结果相对是更准确的。

总结： 1. 唯一捷径：早开始和方法正确不走弯路。 画好 = 1。 方法正确 + 2. 长期坚持 2. 做任何事情前，都要考虑试错成本。 3. 减少试错成本的方法是：跟随正确引导，分阶段学习，在些基础上，某阶段的定点重复学习。

3. 学绘画的方法

阶段学习

学画画的方法：两个字，——拆分，也就是分阶段学习。

遇到一件难事，特大，特难，特复杂，怎么解决？有经验的人都明白，拆分开来，一步一步逐个攻破。如果拆分了还是很难呢？那你拆分的还不够呢。

如果已经拆分了阶段学习，那学到什么程度进入下一个阶段？ 已经有无数文章为此争论不休，但都没有给出解决方法。

争论点

阵营左：长期苦练基础，扎实打牢一步步来，基础不牢创作都是空谈。

数据显示：90%的学习者出现跨阶段，上来就创作，但透视，人体结构，构图，黑白关系，色彩等等全是问题。原因：急！！！！急于表达，急于工作。
但三年人体，十年速写。听上去就吓人。尤其是很多价格昂贵却只能都基础的班，老师都说基础太重要了，不画3、5年，后面的别去想。

阵营右：绘画源自兴趣与表达，能跨阶段就要跨，从实践中学习。

是的，这个论点也有点道理。绘画很感性，是人的兴趣与表达，想画个萌妹子。你告诉我先画2年人体基础？想画个大战舰，你告诉我先画2年盒子，练透视结构？2年之后再来画萌妹子和战舰？？拜托，没有创作的热情，怎么画下去？又有一批价格昂贵大唱速成论的班说：零基础几个月就能创作。

发现没？

基础练不好确实可以创作，但效果甚微！ 而如果长期练基础，又压抑创作欲望，安不下心。 两者为因果关系，两种方式同一时间内不可兼得。要！取！舍！

并且每个阵营后面还有一帮为自己立场吆喝的谋利培训机构。 这成了很多人的困惑。

解决方案很简单，那就是测试

画的好坏有标准吗？当然有！，有一系列明确的指标：

漫画、插画、游戏、影视，都是就业的工种，都由市场大众来衡量，都有明确的指标，面我们学习绘画的内容，就是逐个达到这些指标。 一张画“感觉”各有不同，无标准？这和之前说的“天赋”一样，都是金字塔顶端的事，我们还远没到那份上。

学习绘画的正确方法：是上来就创作，当然可以。但同时有阶段测试的存在。通不过，就说明基础还不足已经限制创作，要做选择，回去按阶段弥补或接着创作。 目标是画好的人就会回去按阶段弥补，为将来画更好的创作做准备。 目标是创作的人就接着创作，不会进步，或者说进步甚微，但释放表达欲。

创作 -> 创作不好 -> 回去学基础 -> 再来创作 -> 再发现创作不好 -> ... 有了阶段测试，随时检测自己的能力，便可以两者间不断切换，交替前进。 某一时间段内，怎么选择，没有任何老师，机构能支配你，只有自己想通才行，因为每个人的性格，目标需求，家境，等均不同。

任何成熟的技术职业培训，学校中的考试都必然是接轨就业的，比如：医生，律师，厨师，等等。

减少试错成本的方法刚好是学习绘画的方法：拆分 1. 按计划，分阶段练习。 2. 在 1 的基础上，某阶段的定点重复学习！ 3. 通过测试来检测阶段成果。 4. 打牢基础再创作，和基础不牢进行创作，两种方式同一时间内不可兼得，要！取！舍！因人而异，自己调控。

学绘画的动力与主动学习

学习绘画的动力

学习绘画一定会遇到困难和瓶颈，要能长期坚持。本质就是在于动力的多少。每个人学画的动力不同。有兴趣热爱，赚钱，出名，也有好胜心。
有些人未必多爱画画，但他们好胜，做事就是想做得比你好，高手大多性格好胜，相反，安逸性格，哪怕秀爱画画却未必能抗住长期坚持。

性格要强的真爱画画的，哪怕自身和外在条件多么差，都会玩命画，最终成功。
性格懒散的，过分安于现状的，即使自身和外在条件多么好，没法坚持不能克服困难，最终也是半吊子。

因此，

要开始漫长学画之路，请先问自己的动力是否足够充分。兴趣要，但不足够，且兴趣易变，很难长远。好胜心，性格，骨子里的东西，反而更重要！！！

主动学习

是自己做出选择，选择想学的内容去学，学习方式是探索，要有积极性。

扪心自问，在学习过程中你是否愿意将自己无法解决，并且不成熟的作品展示给别人，请求指点？并向任何一个比你优秀的人提问？即使被冷眼，仍然接着去求教？

被动学习

别人让（逼）你学，你未必想学的内容，学习方式是躺学，没积极性。

什么是躺学：被动接受别人安排，学习内容和学到什么程度，都是别人给你安排好，按标准通关就可以。更不需要主动提出质疑。寻找方向。 就像孩子被父母喟饭，不用去思考为什么要吃，能不能选择吃其他东西？吃到什么程度算饱！
主动学习要做的事
1. 主动

重点“主” ！—— 自己！！！

自已的主动，看向自己内心深处，分析动力的成分，真爱？名利？好胜心？是否足够？

从此没有逼你，没人限制你，你可以学一年，也可买学2年，可以选择左，或者右，一切自己决定，自己承担后果。 至于“动” 行动！！！ 道理都懂，就剩行动来做了。
  1. 展示不足

传统的教育方式是：

  • 等老师上谭，不会主动找老师提问题
  • 等收作业，不会主动找老师解决作业中的疑问

这两种理念铸造了很多学生在离开学校自我学习绘画时，出现即使老师在身边，也不会该去提问，一味等待老师主动教授。躺习惯了，等喂饭，或者遇到问题，往往是怕老师批评，怕同学笑话，把问题藏起来，而不愿第一时间将自己的不完美展示给老师，浪费大量时间。
主动学习！请放下那份“自尊心”，“三人行，必有我师”，积极展示不足，这反而是强者必备。

  1. 质疑，多问

质疑精神是必要的学习态度，绘画的标准从来都不是唯一的，老师给的答案不一定是唯一的答案。
凡事都应该问个为什么？凭什么？关系到自己知识正确积累更当如此。
质疑的人多问，自然更进步。

每天都思考，我的目标是什么？现在学习这个内容对我要达到的目标有帮助吗？怎么学？和谁学？凭什么和他学？学多久？学到什么程度？等等。但怎么获得质疑的答案呢？多方比较，咨询朋友。或者比你优秀的人。

如果你真的在乎自己的时间，精力，那么你会这么做。

总结：

1. 学习绘画需要多少年？ 2. 学习分阶段 3. 对自己民的学习成果进行检测，最好按阶段循序渐进。 4. 主动！分析动力与行动，展示不足，质疑多问。
