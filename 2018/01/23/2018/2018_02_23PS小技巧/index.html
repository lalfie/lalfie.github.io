PS小技巧

Jan 23, 2018 | PS

混合模式

叠加

  • 白色叠加：可以使原来的颜色更浅（饱和度更低，明度更高）
  • 黑色叠加：可以使原来的颜色更深（饱和度更高，明度更低）

高反差保留原理

把高斯模糊图层反向，混合模式改为线性光，并改为百分之五十。
我们会发现它等于我们的高反差保留图层
由此我们可以推论出
原始图片 - 高斯模糊 = 高反差保留
*原始 = 高反差保留 + 高斯模糊 *
这就韵味着我们可以将任何一幅图像分成两部分：

  • 高斯模糊部分
  • 高反差保留部分
  • 两部分互相补充形成

高低频的作用

低频层可以用来调节图像整体的调子，也就是拉平图像，打散色块。起到的效果和渐层是一样的。这里的调节不会影响到任何图片的细节。细节都在高频层，可以用图章和修复工具，由于没有颜色的影响，这里操作只改变细节，不改变颜色。

简单的说就是一个作用是锐化，另一个作用是磨皮（模糊）

高低频磨皮

操作方法

  • 关闭观察组，在操作图层里盖印图层（Ctrl + shift + Alt + E)。
  • 反向
  • 混合模式改为线性光
  • 执行 滤镜 –> 其它 –> 高反差保留
  • 高反差保留数字为3的位数
  • 再执行 滤镜 –> 模糊 –> 高斯模糊
  • 高斯模糊的参数值为高反差保留参数的三分之一
  • 使用硬度为 0 的大画笔，把五官和明暗交界线擦出来。只柔和皮肤，不能对五官进行柔和。

USM 锐化

作用：增加纹理细节。
操作方法：

  • 数量加大半径，减小是锐化纹理度。反之则是增加对比度。

注意：减小对皮肤的锐化，加大对五官的锐化，锐化不能过死。
